# ROSMASTER R2 コントローラー操作仕様書

本ドキュメントは、Yahboom製 ROSMASTER R2を付属のワイヤレスコントローラーで操作するための仕様をまとめたものです。

---

## 1. 基本操作（操作モードの切り替え）
台車を動かすには、まず**操作モードを有効化**する必要があります。

* **R2ボタン（ボタン番号9）**: **操作モードの有効/無効 切り替え**
    * 有効化するとターミナルに `Joystick Mode: Enabled` と表示され、スティック操作が可能になります。

---

## 2. ボタン・スティック機能一覧

| 操作カテゴリ | 操作デバイス | 対応ボタン番号 | 実行される機能 |
| :--- | :--- | :---: | :--- |
| **システム** | **R2ボタン** | 9 | **操作モードのON/OFF（安全ロック）** |
| | **STARTボタン** | 11 | 本体ブザー（Beep音）の鳴動 |
| **移動制御** | **左スティック** | 0, 1 | **台車の移動**（前後進・左右スライド） |
| | **右スティック** | 2 | **台車の旋回**（左右ターン） |
| | **右スティック押し込み** | 13, 14 | 速度ギヤの切り替え（低速 / 通常） |
| **カメラ(雲台)** | **十字キー** | 6, 7 (Axis) | カメラの向き調整（上下左右） |
| | **R1ボタン** | 7 | カメラの向きを初期位置（正面）にリセット |

---

## 3. 詳細仕様

### 移動（左・右スティック）
* **前進/後退**: 左スティックを上下に倒す。
* **並行移動**: 左スティックを左右に倒す（メカナムホイールによる横滑り走行）。
* **旋回**: 右スティックを左右に倒す。その場で回転します。

### 速度調節（ギヤチェンジ）
* **右スティックを押し込む**たびに、最高速度の制限（ギヤ）が切り替わります。
    * **Linear Gear**: 直進速度を 1/5 に制限します。
    * **Angular Gear**: 旋回速度を 1/4 に制限します。
    * 狭い場所で精密に動かしたい時に使用します。

### カメラ制御
* **十字キー**: ステップ単位でカメラ角度を動かします。
* **R1ボタン**: どこを向いていても、一瞬でカメラを正面位置に戻します。
    * ※環境変数 `INIT_SERVO_S1/S2` で設定された角度に準拠します。

---

## 4. 実行時の注意
* プログラム実行時に `/dev/input/js0`（コントローラーのレシーバー）が認識されている必要があります。
* 操作が効かない場合は、コントローラー中央のモードランプが正しく点灯しているか確認してください。
